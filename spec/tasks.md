- **验收**: 输出DataFrame包含28列特征; 无NaN; 特征值范围合理

## 迭代2: 核心算法引擎 (Phase 2)

### Task 2.1: Agri-PC因果发现算法 (agri_pc.py)
- **优先级**: P0 | **预估**: 90min
- **内容**:
  1. PC算法骨架: 完全图构建→条件独立性检验(Fisher-Z)→逐步删边→V结构识别→Meek规则
  2. 时序偏序约束: T(i)<T(j)时移除违例边(原因必须先于结果)
  3. 农业周期先验: 强制保留"播种→产量→价格"因果链
  4. 交割约束: 强制保留"现货→期货交割期"边
  5. DAG质量评估: F1-score计算(与真实DAG对比), 搜索空间缩减率
  6. 因果链高亮: 识别并高亮核心因果路径
  7. NetworkX DiGraph输出
- **验收**: DAG F1-score≥0.85(目标0.89); 搜索空间缩减率≥20%(目标21.9%); 运行≤37.2s

### Task 2.2: ACML因果元学习器 (acml.py)
- **优先级**: P0 | **预估**: 90min
- **内容**:
  1. 部分线性模型: Y = g(X) + τ(X)·D + ε
  2. 时间序列K折分割: K=5, 保持时序结构
  3. 第一阶段-处理模型: XGBoost拟合D=f(X), 残差D̃=D-f̂(X)
  4. 第一阶段-结果模型: XGBoost拟合Y=g(X), 残差Ỹ=Y-ĝ(X)
  5. 第二阶段-CATE估计: Ỹ = τ(X)·D̃ + ν
  6. 双重正交化验证: 残差正交性检验
  7. Neyman正交性验证: 扰动nuisance参数检验稳定性
  8. 农业风险正则项: λ×E[Risk×(τ(X)-τ̄)²]
  9. CATE函数输出: τ̂(X)可调用对象
- **验收**: 极端天气误差率较T-Learner降低≥45%(目标51%); 训练≤12.5s; MAPE≤1%

### Task 2.3: CCP因果保形预测 (ccp.py)
- **优先级**: P0 | **预估**: 80min
- **内容**:
  1. 因果残差计算: r_i = Y_i - τ̂(X_i)·D_i - ĝ(X_i)
  2. 非一致性分数: s_i = |r_i| / σ̂(X_i)
  3. 分位数计算: q = ceil((1-α)(n+1))/n
  4. 预测区间构建: [ŷ - q·σ̂(X), ŷ + q·σ̂(X)]
  5. 自适应覆盖调整: α_{t+1} = α_t + γ(1 - 1{Y_t ∈ Ĉ_t})
  6. 加权降权: 历史分布偏移样本权重降低
  7. Walk-Forward验证: 8窗口滚动验证
- **验收**: 分布偏移下覆盖率≥88%(目标90%); 预测≤0.8s; 自适应5步内收敛

### Task 2.4: 基准模型 (baselines.py)
- **优先级**: P1 | **预估**: 40min
- **内容**:
  1. XGBoost基准: XGBRegressor直接预测
  2. ARIMA基准: statsmodels ARIMA(p,d,q)
  3. 传统精算基准: 历史损失率×风险系数
  4. S-Learner: 单模型CATE估计
  5. T-Learner: 双模型CATE估计
  6. DML: EconML LinearDML
  7. IV: 两阶段最小二乘(2SLS)
- **验收**: 所有基准模型可正常运行; 输出格式统一

### Task 2.5: 实验验证引擎 (validation_engine.py)
- **优先级**: P1 | **预估**: 50min
- **内容**:
  1. Walk-Forward验证: 8窗口×36品种
  2. 五重因果验证: PSM/S-Learner/T-Learner/DML/IV
  3. 消融实验: 9创新点逐个消融
  4. 纯预测验证: 含lag vs 不含lag + Clark-West检验
  5. 四川省特色验证: 生猪/油菜籽/柑橘
  6. 极端灾害验证: 2024河南暴雨
  7. 社会价值测算: 全国+四川效益
